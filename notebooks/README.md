# Bluestock Mutual Fund Analytics - Notebooks Directory

This directory contains Jupyter Notebooks used for Exploratory Data Analysis (EDA), prototyping data cleaning functions, and early visualization testing before logic was migrated into production Python scripts.

## Contents
- **`EDA.ipynb`**: Initial exploration of historical NAV data, identifying missing values, data distributions, and correlation between funds.
- **`Performance_Analytics.ipynb`**: Scratchpad for prototyping the CAGR, Volatility, Sharpe, and Sortino calculations logic.
- **`reports/`**: Output directory for visualizations generated by both notebooks and scripts (e.g., `efficient_frontier.png`, `monte_carlo_projection.png`).

To run these notebooks, ensure you have Jupyter installed and start the server from the project root:
```bash
jupyter notebook notebooks/
```
