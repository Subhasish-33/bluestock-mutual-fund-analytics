# Bluestock — Mutual Fund Analytics
**Capstone Project I | Data Analyst Internship 2026**

## Project Overview
End-to-end data analytics pipeline on mutual fund NAV data,
fund performance metrics, and investor insights for Bluestock Fintech.

## Tech Stack
Python · Pandas · NumPy · Matplotlib · Seaborn · Flask
PostgreSQL · SQLAlchemy · yfinance · mfapi.in

## Project Structure
data/raw/        → raw CSVs (not committed)
data/processed/  → cleaned datasets
notebooks/       → EDA and analysis notebooks
sql/             → PostgreSQL queries
dashboard/       → Flask API
reports/         → final report
